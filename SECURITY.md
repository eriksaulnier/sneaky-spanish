# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Sneaky Spanish, please report it through one of these channels:

- **GitHub Issues** — for non-sensitive issues, [open an issue](https://github.com/eriksaulnier/sneaky-spanish-v2/issues/new) with the label `security`
- **Email** — for sensitive disclosures, reach out via the contact info on [my GitHub profile](https://github.com/eriksaulnier)

## Scope

This policy covers the extension code in this repository. It does not cover the websites the extension runs on.

Examples of in-scope issues:
- XSS through translated word content
- Data leaking from `chrome.storage`
- Content script accessing or modifying sensitive page data unintentionally

## Response

I'll acknowledge reports within a few days and aim to release a fix promptly. If you include a suggested fix, even better.
